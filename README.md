# The Unix Shell Interpreter: ``sh`` shell.

## A team implementation of a custom ``sh shell``

## Compiling:
Program will be compiled this way:

```shell
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh
```
